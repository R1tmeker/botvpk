# Telegram Mini App

Статичная заготовка визуального интерфейса для ВПК-бота.

Чтобы кнопка Mini App появилась в Telegram:

1. Опубликуйте папку `miniapp/` на публичном HTTPS-хостинге.
2. Укажите URL в `config.txt` как `MINI_APP_URL=https://...`.
3. Перезапустите бота.

Бот добавляет к URL параметр `role`, поэтому интерфейс показывает доступные разделы под роль пользователя. Выбранный раздел отправляется через `Telegram.WebApp.sendData`, после чего бот открывает соответствующий раздел обычным сообщением.

Иконки разделов лежат в `assets/icons/` в формате PNG и сделаны под тёмно-синюю палитру проекта.
